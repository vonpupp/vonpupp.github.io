Vonpupp’s top knots #

This document is a summary of the knots that I consider useful for bushcraft and survival. This document is in fact crafted to be presented as an Anki deck that can be downloaded here.

Be aware that the Anki deck information is presented in Portuguese, while the Org core information is in English (hence the english/portuguese mess)
